# Pawpid: Where Tails Wag and Hearts Connect

**Pawpid** is a web-based **Multi-Shelter Adoption Matching System** that connects adopters with stray animals through intelligent, data-driven recommendations.  
Built to support animal shelters and responsible adopters in the Philippines, the system addresses low adoption rates by enabling more accurate and humane matches.
